50,000-won note
First issuance June 23, 2009
Size (mm) 154×68
Basic color Yellow
Material Cotton
Themes Shin Saimdang (1504~1551), Mukpododo, Egg plants of Chochungdo-subyeong, Wolmaedo, Poongjukdo
Security Holographic Strip, Moving image in the striped band, Color Shifting Ink, Intaglio Latent Image, Security Thread, See-Through Register, Micro Lettering, Watermark, Special Press and Soldering, Intaglio Printing, Fluorescent Security, Fibers, etc.
